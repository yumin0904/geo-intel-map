## 핵심 주장

Olson & Zeckhauser(1966)는 집단 방어 동맹을 **공공재 문제**로 재정식화했다:

> "집단 방어에서 억지력(deterrence)은 비배제성·비경합성을 가진 순수 공공재다. 따라서 대국이 과다 공급하고 소국은 무임승차한다."

### 무임승차의 구조적 원인

```
억지력(공공재) = 비배제 (NATO 회원국 모두 혜택)
              + 비경합 (A국 소비가 B국 소비 감소 안 됨)
              → 소국 기여 인센티브 약화 → 무임승차
```

### NATO 방위비 분담 실태 (2023 SIPRI)

| 국가 | GDP 대비 방위비 | 2% 기준 충족 |
|------|-------------|------------|
| 미국 | 3.4% | ✅ 초과 |
| 폴란드 | 4.0% | ✅ 초과 (러시아 위협 근접) |
| 그리스 | 2.5% | ✅ |
| 독일 | 1.57% → 2.1%(2024) | ✅ 최근 달성 |
| 이탈리아 | 1.5% | ❌ |
| 스페인 | 1.3% | ❌ |

→ 2024년 기준 32개 회원국 중 23개 2% 달성 (사상 최고)

### Sandler의 '순수-혼합 공공재' 구분

초기 Olson-Zeckhauser는 억지력을 순수 공공재로 가정 → 무임승차 극단 예측.  
실제 군사력은 **혼합 공공재(impure public good)**:
- 전방 배치 병력 = 소국 이익에도 직접 기여 → 기여 인센티브 일부 복원
- 특정 지역 방어(예: 발트 3국) = 해당 소국에만 주요 혜택 → 클럽재(club good) 성격

### 트럼프 효과: 2% 압박의 정치화

트럼프 1기(2017~2021)·2기(2025~) 방위비 압박:
- "방위비 미달 국가는 NATO 보호 보장 못한다" → 동맹 조건부화 위협
- 폴란드·발트국 방위비 급증 vs 독일·프랑스 저항 → 동맹 내 분열 심화

## 현재 사례 연결

- **독일 방위비 2% 달성(2024)**: 무임승차 청산의 역사적 전례 — Zeitenwende(전환점) 효과
- **한국 방위비 분담**: 매 5년마다 재협상 → 비대칭 동맹의 비용분담 협상 전형
- **일본 방위비 2배 증액(2022~2027)**: GDP 2% 목표 → 아시아 버전 NATO 2% 논쟁
